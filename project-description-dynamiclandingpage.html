DYNAMIC LANDING PAGE

Project description Dynamic Landing Page

Background

This project is a try-out to use a combination of beautiful pictures from pexels.com and JS.

Dynamic Landing Page is re-usable in all kinds of ways. There are two pieces of the interface that can be modified by the end-user. The modified values are stored into local cache.

I'd like to thanks Pexels.com for supplying the nice images for this project.
