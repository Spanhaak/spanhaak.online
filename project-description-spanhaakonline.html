Project description spanhaak.online

Background

Since a while I was thinking of how best to have an online platform which I could use as a showcase of some of my projects and also give people the ability to get to know me a bit better.

In the about section, there is a lot of information about me and there you can download my extended CV and also an overview CV in one page. If requested I can also provide a list of projects which I run or participated in.

Aside from the projects I would like to explain also what the purpose was of each of these projects and also which technology was used to build them.

I like clean and simple web pages, although there are a couple of nice features under the hood, I dont want to have unnecessary 'bells and whisles' on this site.
